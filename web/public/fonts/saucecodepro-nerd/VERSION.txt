------------------------------

     SourceCodePro patched ttf files (regular + bold).
  2. Subset each to latin / latin-ext / PUA codepoints with
     pyftsubset (fonttools).
  3. Convert each subset ttf to woff2 with woff2_compress
     (google/woff2).
  4. Drop the resulting woff2 files in this directory and update the
     src: chain in saucecodepro-nerd-font.css to start with their
     url(...) entries (system local() fonts then become a graceful
     fallback while the woff2 loads).
  5. Record the SHA-256 of every woff2 file below so future bumps are
     verifiable.
